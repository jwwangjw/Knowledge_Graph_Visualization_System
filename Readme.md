# Readme.md

## 关于项目

项目为软工三大作业知识图谱可视化项目，前端使用vue.js框架，后端使用springboot+mybatis，数据库使用mysql。项目主要实现了类似neo4j图数据库的知识图谱可视化系统，并且可以实现简单的知识抽取和知识融合。

## 项目部署
将代码进行下载，前端需要node.js包管理系统，运行npm run build生成dist文件夹，后端直接使用jar包，将数据库和nginx进行相应配置后即可访问。
